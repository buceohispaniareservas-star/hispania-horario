⚙️ CONFIGURAR FIREBASE
1. Ve a console.firebase.google.com → Nuevo proyecto.
2. Activa Realtime Database → Modo prueba.
3. Ve a Configuración del proyecto → SDK web → copia los datos.
4. Abre este HTML, busca FIREBASE_CONFIG y pega tus datos reales.
5. En Firebase → Rules:
{ "rules": { ".read": true, ".write": true } }
6. Sube el HTML a GitHub → Settings → Pages → activa.
